# QPESK JUSE Benchmarking Redesign

사단법인 한국품질기술사회(QPESK) 홈페이지 개편 시안입니다.

## 방향

- 기존 QPESK 홈페이지의 비전, 미션, 핵심가치, 플랫폼, 회원가입 안내를 보존
- JUSE 구조를 참고해 교육, 현장 품질혁신, 품질기술경영대상, 세미나, 아카이브, 출판 중심으로 재배치
- ASQ/CQI처럼 회원, 기업교육, 기술자료, AI 품질혁신 허브로 확장 가능한 구조 반영

## 파일

- `index.html`: 홈페이지 시안
- `styles.css`: 반응형 디자인 스타일
- `script.js`: 모바일 메뉴 동작

## 확인 방법

`index.html` 파일을 브라우저로 열면 됩니다.

이 시안은 원본 `qpesk.com`을 수정하지 않는 로컬 프로토타입입니다.
