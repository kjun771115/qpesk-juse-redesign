# QPESK JUSE Benchmarking Redesign

사단법인 한국품질기술사회(QPESK) 홈페이지 개편 시안입니다.

## 방향

- 기존 QPESK 홈페이지의 비전, 미션, 핵심가치, 플랫폼, 회원가입 안내를 보존
- JUSE 구조를 참고해 교육, 현장 품질혁신, 품질기술경영대상, 세미나, 아카이브, 출판 중심으로 재배치
- ASQ/CQI처럼 회원, 기업교육, 기술자료, AI 품질혁신 허브로 확장 가능한 구조 반영

## 파일

- `index.html`: 홈 랜딩 페이지
- `about.html`, `education.html`, `field.html`, `award.html`, `archive.html`, `ai.html`, `join.html`: 메뉴별 전용 페이지
- `styles.css`: 반응형 디자인 스타일
- `script.js`: 전체화면 메뉴, 아코디언 하위 메뉴, 페이지 내 서브내비 동작

## 메뉴 구조

ASQ(asq.org)를 벤치마킹해 상단 메뉴는 햄버거 버튼을 누르면 전체화면 슬라이드 메뉴가 열리고,
하위 항목이 있는 메뉴는 `+` 버튼으로 그 자리에서 펼쳐지거나 이름을 누르면 해당 전용 페이지로
이동합니다. 각 전용 페이지에는 다시 하위 탭(서브내비)이 있어 세부 콘텐츠로 이동합니다.

## 확인 방법

`index.html` 파일을 브라우저로 열면 됩니다.

이 시안은 원본 `qpesk.com`을 수정하지 않는 로컬 프로토타입입니다.
